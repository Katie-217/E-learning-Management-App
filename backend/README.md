# Backend (Skeleton)

Suggested stack: Node.js + TypeScript (Express or NestJS) with PostgreSQL/MySQL/MongoDB.

## Structure
```
backend/
  src/
    config/
    models/
    controllers/
    routes/
    services/
    middleware/
    utils/
  tests/
  server.ts
```

## Development
- Use Docker for DB in local.
- Provide `.env` with DB connection and JWT secrets.
